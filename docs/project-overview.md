# Project Overview

## 1. Project Purpose

Match-4 is a simple web-based puzzle game that combines "Match-3" swapping mechanics with "Color Lines" matching rules (4 in a line). The goal is to provide a fun and engaging user experience with smooth animations.

## 2. Executive Summary

This project is a standalone, client-side single-page application (SPA) built with modern web technologies. It has no backend and manages its state in the browser.

## 3. Technology Stack Summary

| Category      | Technology     |
|---------------|----------------|
| **Frontend**  | React, TypeScript, Vite |
| **Animation** | Framer Motion  |
| **Testing**   | Vitest         |

For more details, see the [Architecture Documentation](./architecture.md).
